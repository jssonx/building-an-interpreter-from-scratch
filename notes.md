# Notes

### 下面三个表达式等价
```javascript
function functionName(param1, param2, ...) {
  // code to be executed
}
```

```javascript
let functionName = (param1, param2, ...) => {
  // code to be executed
}
```

```javascript
let functionName = function (param1, param2, ...) {
  // code to be executed
}
```
